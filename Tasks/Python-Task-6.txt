-------------------------------------------------- SQL – ALTER Commands --------------------------------------------------

-------- Objective: Practice modifying the structure of an existing table using ALTER commands --------

1. Create a database named company_db.

    CREATE DATABASE company_db;
    USE company_db;

2. Create a table named employees with the following columns

    CREATE TABLE employees(
    emp_id INT PRIMARY KEY,
    emp_name VARCHAR(50),
    salary FLOAT
    );

3. Perform the following ALTER operations:

    * Add a new column department VARCHAR(30)
        ALTER TABLE employees
        ADD department VARCHAR(30);

    * Modify the salary column datatype
        ALTER TABLE employees
        MODIFY salary DOUBLE;

    * Rename column emp_name → employee_name
        ALTER TABLE employees
        RENAME COLUMN emp_name TO employee_name;

    * Drop the column department
        ALTER TABLE employees
        DROP COLUMN department;

    * Rename the table employees → staff
        RENAME TABLE employees TO staff;

4. Write queries to

    * Drop the table staff
        DROP TABLE staff;
    
    * Drop the database company_db
        DROP DATABASE company_db;



-------------------------------------------------- Python – Error Handling --------------------------------------------------

-------- Objective: Understand different types of errors and handle exceptions in Python --------

1. Write a Python program that divides two numbers and handles.
    
    * ZeroDivisionError & ValueError

        try:
            a = int(input("Enter first number: "))
            b = int(input("Enter second number: "))
            
            result = a / b
            print("Result:", result)

        except ZeroDivisionError:
            print("Cannot divide by zero")

        except ValueError:
            print("Invalid input. Please enter numbers only")

        OUTPUT : * Enter first number: 0
                   Enter second number: 10
                   Result: 0.0
                 
                 * Enter first number: 10
                   Enter second number: 0
                   Cannot divide by zero

                 * Enter first number: a
                   Invalid input. Please enter numbers only

2. Write a program demonstrating the use of.
    
    * try & except & finally

        try:
            num = int(input("Enter a number: "))
            print("Number entered:", num)

        except ValueError:
            print("Input should be a number")

        finally:
            print("Executed")

    OUTPUT : * Enter a number: sagar
               Input should be a number
               Executed

             * Enter a number: 33      
               Number entered: 33
               Program execution completed



-------------------------------------------------- Python – File Handling --------------------------------------------------

-------- Objective: Practice basic file operations in Python --------

1. Create a file named data.txt and write some text into it.

    file = open("data.txt", "w")
    file.write("This is my first file.\n")

2. Append additional content to the same file.

    file = open("data.txt", "a")
    file.write("Adding more content to the file.\n")

3. Read the entire file content using .read().

    file = open("data.txt", "r")
    content = file.read()
    print(content)

    OUTPUT : This is my first file.
             This is an appended line.

4. Read the file line by line using .readline() or .readlines().

    file = open("data.txt", "r")
    lines = file.readlines()
    for line in lines:
        print(line)

    OUTPUT : This is my first file.

             This is an appended line.

5. Close the file after completing the operations.

    file.close()

-------------------------------------------------- SQL – Subqueries --------------------------------------------------

-------- Objective: Understand how to use queries inside other queries to retrieve data. --------

1. Create a database named subquery_db.

   CREATE DATABASE Subquery_db;
   USE Subquery_db;

2. Create a table named employees with the following columns:

   CREATE TABLE Employees(
   Emp_Id INT PRIMARY KEY,
   Emp_Name VARCHAR(50),
   Salary FLOAT,
   Department VARCHAR(50)
   ); 

3. Insert at least 6 records into the table.

   INSERT INTO Employees VALUES
   (1,'Sagar',50000,'IT'),
   (2,'Deepika',60000,'HR'),
   (3,'Lakshman',35000,'IT'),
   (4,'Nikhil',45000,'Finance'),
   (5,'Naveen',40000,'HR'),
   (6,'Manjusha',30000,'IT');

4. Write SQL queries demonstrating the following types of Subqueries:

   * Single Row Subquery
     
     > Find employees whose salary is greater than the average salary
       
       ---> SELECT Emp_Name, Salary
            FROM Employees
            WHERE Salary >
            (SELECT AVG(Salary) FROM Employees);

            OUTPUT :    Emp_Name    Salary  
                        Sagar	    50000
                        Nikhil	    45000
                        Deepika     60000

   * Multiple Row Subquery

     > Use operators such as:

       ---> IN

            SELECT Emp_Name, Department
            FROM Employees
            WHERE Department IN
            (SELECT Department FROM Employees WHERE Department='HR');

            OUTPUT :    Emp_Name    Department
                        Deepika	        HR
                        Naveen	        HR

       ---> ANY

            SELECT Emp_name, Salary
            FROM Employees
            WHERE Salary > ANY
            (SELECT Salary FROM Employees WHERE Department='HR');

            OUTPUT :    Emp_name    Salary
                        Sagar	    50000
                        Deepika	    60000
                        Nikhil	    45000
            
       ---> ALL

            SELECT Emp_Name, Salary
            FROM Employees
            WHERE Salary > ALL
            (SELECT Salary FROM Employees WHERE Department='IT');

            OUTPUT :    Emp_Name    Salary
                        Deepika	    60000

   * Correlated Subquery
     
     > Write a query where the inner query depends on the outer query

       ---> SELECT Emp_Name, Salary, Department
            FROM Employees E1
            WHERE Salary >
            (
            SELECT AVG(Salary)
            FROM Employees E2
            WHERE E1.Department = E2.Department
            );

            OUTPUT :    Emp_Name    Salary    Department
                        Sagar	    50000	      IT
                        Deepika	    60000	      HR



-------------------------------------------------- Python – Strings --------------------------------------------------

-------- Objective: Practice string operations and built-in string methods. --------

1. Write a short note explaining:

   * What is a String
     > String is a sequence of characters enclosed in quotes.
       It can contain letters, numbers, or symbols.

       ---> Example : Name = "Sagar"

   * String properties:
     
     > Iterable  : Characters in a string can be accessed one by one using loops.
     > Immutable : String values cannot be changed once they are created.
     > Ordered   : Characters in a string have a fixed order and index position.

2. Demonstrate String Operations:

   * Concatenation
     > str1 + str2
       
       ---> str1 = "Hello"
            str2 = "World"

            result = str1 + " " + str2
            print(result)

            OUTPUT : Sagar Chowdary

   * Repetition
     > str1 * 3

       ---> text = "Sagar Chowdary "
            print(text * 3)

            OUTPUT : Sagar Chowdary Sagar Chowdary Sagar Chowdary

3. Demonstrate the following Built-in String Methods:

   ---> Name : sagar Chowdary

        * .upper()
            
           print(Name.upper())

           OUTPUT : SAGAR CHOWDARY

        * .lower()

           print(Name.lower())

           OUTPUT : sagar chowdary
        
        * .title()

           print(Name.title())

           OUTPUT : Sagar Chowdary

        * .capitalize()

4. Demonstrate Whitespace Removal Methods:
   
   ---> Name = "   Sagar Chowdary   "
   
        * .strip()
           
           print(Name.strip())

           OUTPUT : Sagar Chowdary

        * .lstrip()

           print(Name.lstrip())

           OUTPUT : Sagar Chowdary

        * .rstrip()

           print(Name.rstrip())

5. Demonstrate Other String Methods:

   ---> Name = "Vidya Sagar"

        * .replace(old, new)

           OUTPUT : Kota Sagar

        * .split(breakChar) → splits string into multiple substrings stored in a list

           OUTPUT : ['Vidya', 'Sagar']

        * .join(iterable) → joins elements of iterable into a string

           print(" ".join(["Vidya", "Sagar"]))
